# 残業時間管理アプリ

スマホでの利用を想定した、ブラウザだけで動く残業時間管理アプリです。
残業した日に退勤時刻を記録する前提で作っています。

## 使い方

1. `index.html` をブラウザで開きます。
2. 退勤時に「退勤打刻」を押します。
3. 同じ勤務日に再度保存する場合は、確認後に上書きします。
4. 残業なしの日も打刻すると、平均の計算対象になります。
5. 直近1年の月別推移と、この期間の記録を棒グラフで確認できます。
6. Supabaseへログインすると、端末保存の記録をクラウドにも同期します。

## スマホで開く

会社で使うには、GitHub PagesやNetlifyなどで公開URLにしてからスマホで開きます。
未ログイン中はこの端末に保存し、Supabaseへログインするとクラウドにも同期します。

公開後はスマホのブラウザでURLを開き、ホーム画面に追加して使います。

## 集計ルール

- 集計期間は16日から翌月15日までです。
- 残業は17:00以降です。
- 17:00から17:15、19:15から19:30は休憩として除外します。
- 残業は1分単位で、秒は切り捨てます。
- 平均は打刻した日数で計算します。未打刻日は休日として扱います。
- 保存・表示する記録は直近1年分です。それより古い記録は削除します。
- 現在期間と前期間は修正できます。2カ月前になった期間は確定済みとして、一覧や修正・削除を表示しません。
- 17時前の退勤は、通常は残業0分として扱います。
- 深夜帯は「残業ですか？」の確認で、はいなら前日分の残業、いいえなら0分として記録します。
- 休日出勤は「特殊な勤務」から選んで記録します。
- 早出は出勤時に「特殊な勤務」から「早出」を選び、「出勤打刻」を押します。退勤時に早出分を自動で加算します。
- 休日出勤は8:35開始、10:00〜10:15、11:45〜12:30、15:00〜15:15、17:00〜17:15、19:15〜19:30を休憩として計算します。
- 土日祝も、記録を付けた日は集計対象です。

## 保存先

未ログイン中の記録はブラウザのローカルストレージに保存します。Supabaseへログインすると、ログインしたユーザーの記録としてクラウドへ同期します。
