# 残業時間管理アプリ

スマホでの利用を想定した、ブラウザだけで動く残業時間管理アプリです。
残業した日に退勤時刻を記録する前提で作っています。

## 使い方

1. `index.html` をブラウザで開きます。
2. 退勤時に「退勤打刻」を押します。
3. 同じ勤務日に再度保存する場合は、確認後に上書きします。
4. 平均に含めたい定時の日は「定時扱い」をチェックして退勤打刻します。
5. 半年の月別推移と、この期間の記録を棒グラフで確認できます。一覧の行を押すと、修正や削除ができます。
6. Supabaseへログインすると、端末保存の記録をクラウドにも同期します。

## スマホで開く

会社で使うには、GitHub PagesやNetlifyなどで公開URLにしてからスマホで開きます。
未ログイン中はこの端末に保存し、Supabaseへログインするとクラウドにも同期します。

公開後はスマホのブラウザでURLを開き、ホーム画面に追加して使います。

## 集計ルール

- 集計期間は16日から翌月15日までです。
- 残業は17:00以降です。
- 17:00から17:15、19:15から19:30は休憩として除外します。
- 残業は1分単位で、秒は切り捨てます。
- 平均は打刻した日数で計算します。未打刻日は休日として扱います。
- 深夜0:00から4:59のワンタップ記録は、前日の勤務として扱います。
- 手入力では、0:00〜16:59の退勤時刻を、選んだ勤務日の残業終了時刻として集計します。
- 土日祝も、残業記録を付けた日は集計対象です。

## 保存先

未ログイン中の記録はブラウザのローカルストレージに保存します。Supabaseへログインすると、ログインしたユーザーの記録としてクラウドへ同期します。
